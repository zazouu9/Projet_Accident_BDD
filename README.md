# Projet_Accident_BDD
Projet BDD sur les accidents de la route.

1) verifier l'etat du git
git status

2) ajouter ou supprimer des fichiers : 
git add/rm [fichiers]

3) enregister les fichiers avec un message
git commit -m "message"

4) envoyer au repo distant
git push



recuperer des fichiers : 
git pull

voire les branche 
git brach 

creer une branche
git branch [nom]

aller sur la branche 
git checkout [nom]

recuperer les infos de main dans la branche au fur et a mesure
git rebase
